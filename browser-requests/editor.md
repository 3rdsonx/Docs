# Introduction

A **Browser Request** is the first feature we have developed for Gaffa and allows you to send our API a simple REST request with the URL of a web page you want to load, a list of actions you want to be carried out and any outputs you want from the page. We'll carry out the request on our browsers and return you the response with no need to worry about proxies, IP rotation, web automation frameworks and scaling.\
\
**We think it's by far the simplest way to automate web tasks and the good news is, we're just getting started and have much more planned.**

In the following pages, we'll walk you through how the browser request API works and how you can integrate it into your solution.
