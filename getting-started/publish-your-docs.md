# Account Setup



{% hint style="info" %}
At this early stage of the Gaffa Browser API, we have created a short pre-approval process to carry out so we can ensure we minimize any unexpected scaling difficulties and can prioritize customers based on our current feature-set.&#x20;
{% endhint %}

In order to start using the Gaffa Browser API you'll need to register for an account and then fill in the short form you'll see after logging in. The form will ask for the following information:

**Usage Description:** a summary of what you plan to use the API for, the more detail the better. Details on what coding language you will be using to call the API will also help us better understand which SDK's we should be using.&#x20;

**Requested URLs:** a list of the sites you want to request using our API. We can't yet provide access to every site on the internet currently so this will help us ensure we can accommodate your requests.&#x20;

**Country:** Add the country you are currently located and help us learn where our customers are based!

After submitting the form, we will review your account and be back in touch with you after we've had a chance to review the information you've submitted and we're ready to support more customers. We hope we'll be able to accommodate as many people as possible but if we can't currently we will let you know and stay in touch when things change.
