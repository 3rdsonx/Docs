# Creating an Account

You can sign up to create a Gaffa account [here](https://accounts.gaffa.dev/sign-up?redirect\_url=https%3A%2F%2Fgaffa.dev%2F%2Fauth%2Fsign-in). After signing up you'll immediately be able to use the API to experiment on [our test site ](https://demo.gaffa.dev/)which simulates a range of automation scenarios. \


**Using Gaffa on the open web**\
In order to avoid scaling issues we are currently restricting access to the API. To expand your use of Gaffa you'll need to join the queue for access and we'll contact you when access is available. If you want to jump up the queue you can fill out a short survey to help us better understand our users. \


**Once your account is approved**\
After we've approved your account (which shouldn't take long) you'll be able to sign up to one of our [paid plans](https://gaffa.dev/#pricing) and you can then start using the API in full.
