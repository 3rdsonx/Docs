# API Reference

Complete HTTP API documentation for Gaffa. Each endpoint page includes interactive OpenAPI definitions you can try from the docs.

Start with [API Authentication](api-authentication.md) to create and use API keys, then explore the endpoints below.

## Browser requests

* [POST v1/browser/requests](post-v1-browser-requests.md) — Create and run a browser request
* [GET v1/browser/requests/{id}](get-v1-browser-requests-id.md) — Get a browser request by ID
* [GET v1/browser/requests](get-v1-browser-requests.md) — List browser requests

## Schemas

* [POST v1/schemas](post-v1-schemas.md) — Create a schema
* [PUT v1/schemas](put-v1-schemas.md) — Update a schema
* [GET v1/schemas](get-v1-schemas.md) — List schemas
* [DELETE v1/schemas/{id}](delete-v1-schemas-id.md) — Delete a schema

## Site mapping

* [POST v1/site/map](post-v1-site-map.md) — Create a site mapping request
* [GET v1/site/map](get-v1-site-map.md) — List site mapping requests
* [GET v1/site/map/{id}](get-v1-site-map-id.md) — Get a site mapping request by ID
